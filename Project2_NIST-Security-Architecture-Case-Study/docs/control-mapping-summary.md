# Control Mapping Summary (High-Level)

This document provides a high-level overview of how key security controls were selected and aligned with organizational risk and compliance requirements.

## Example Mappings

- Access Control
  - RBAC and least privilege aligned with NIST AC family
  - MFA for privileged and remote access

- System Integrity
  - Patch management and vulnerability scanning
  - Endpoint protection controls

- Boundary Protection
  - Firewalls, IDS/IPS, and network segmentation
